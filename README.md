# BetterCAS

## Description
A web app developed for the CITS3403 Agile Web Development group project. This is a calendar and task reminder tool for university assignments and studying, featuring:
- To-do list
- Calendar views
- Sharing calendars with friends

## Instructions

### Launching the App
Clone this repository to your local machine, create a Python virtual environment and run Flask.

```python
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
flask run
```
